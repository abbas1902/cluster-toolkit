# A3-Ultra/A4 High + Slurm + Dynamic Workload Scheduler

This document outlines Slurm support for Dynamic Workload Scheduler (DWS) which is accessible in two modes:

1. [Flex Start](/docs/slurm-dws-flex.md)
2. [Calendar Mode](/docs/slurm-dws-calendar.md)

To use DWS Flex/Calendar modes with other machine types please follow the above guides in order to add the correctly settings to your blueprint.
